Quick Start

Get TunaCode up and running in minutes.

Installation (PyPI) 

pip install tunacode-cli

Development Installation 

# Clone the repository
git clone https://github.com/larock22/tunacode.git
cd tunacode

# Run the setup script
./scripts/setup_dev_env.sh

# Or manually:
python3 -m venv venv
source venv/bin/activate
pip install -e ".[dev]"

One-line Install (Linux/macOS) 

wget -qO- https://raw.githubusercontent.com/alchemiststudiosDOTai/tunacode/master/scripts/install_linux.sh | bash

After installation, configure TunaCode with your preferred LLM provider:

# Direct CLI Setup (Example with OpenAI)
tunacode --model "openai:gpt-4.1" --key "your-openai-key"
